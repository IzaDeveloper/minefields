## How to run locally

### 🧩 Step 1: Clone the repository
```bash
git clone https://github.com/IzaDeveloper/minefields.git
```

### 📁 Step 2: Navigate to the project folder
```bash
cd minefields
```

### 📦 Step 3: Install dependencies
```bash
npm install
```

### ▶️ Step 4: Start the development server
```bash
npm run dev
```
